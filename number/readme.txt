System Design Solution
1. User must input every word for roman number.
2. Everyword that user inputted will be showed at the Alien Symbol table.
3. User can replace the word with input the new one.
4. User can ask Alien with the word that they put before.

Environment
1. Eclipse Oxygen
2. Apache Maven 3.5
3. Apache Tomcat 7

Installation
1. Import the number project to eclipse (using existing maven project)
2. Update project with Maven Update Project
3. Run the project with Maven Build (goals tomcat7:run)
